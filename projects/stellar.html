Collaborated with an all-remote team of six to create a full-stack CRUD interactive prep app to help software developers build confidence for their next technical interview. I implemented Auth and OAuth access delegation in order for users to grant stellar View access to their information on GitHub. I Implemented route handlers in the controller, created a data model to manage data and deal with database interactions. I wrote tests and ran the tests using Jest Test and Super Test. I setup the Postgres database and seeded data for the questions and answers.

This application uses a React, Sass frontend and features flexible authentication. Users can create an account with GitHub or through our in-house authentication forms. Top-reported technical questions are stored in our PostgreSQL database, and fetched from our backend RESTful API server using Express and Node. The backend API uses user routes and quiz routes.
